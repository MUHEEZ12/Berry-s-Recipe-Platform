# Recipe Sharing App — Frontend

Quick steps to run the frontend locally.

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

Open http://localhost:5173 in your browser.

Notes:
- This repo expects `src/main.jsx` and `src` files to be present (already included).
- If you prefer Yarn: `yarn` then `yarn dev`.
